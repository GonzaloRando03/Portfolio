P: Aplicación web Angular, MongoAtlas, Node.js.

Aplicación web orientada a una empresa que se dedica a vender ordenadores y quiere guardar sus datos en una base de datos MongoAtlas.

Al entrar en la página aparece un loggin que al rellenarlo intenta conectarse a la base de datos.

En el menú principal encontramos distintas funciones, como ver las colecciones Ordenadores y Ventas con todos sus objetos, ver la colección Marcas que se crea automáticamente, ver una tabla con las 3 colecciones juntas o ir al menú de consultas.

En la lista de Ordenadores o Ventas podemos ver todos los objetos de la colección, con todos sus campos. En la parte superior encontramos un buscador totalmente funcional que filtra los ordenadores por ID. También encontramos botones de editar y eliminar.

La colección Marcas se crea automáticamente mediante una función que recoge todas las marcas que usamos en los equipos, las contabiliza y nos manda una lista con los equipos en los que aparece.

El Fronted se comunica con un Backend mediante Express y este realiza todas las operaciones con la base de datos.
