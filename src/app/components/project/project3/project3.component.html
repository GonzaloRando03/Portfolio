P: Aplicación de escritorio Python y SQLite.

Aplicación de escritorio programada en Python usando la librería de Tkinter, la aplicación te permite guardar, editar y eliminar registros en una base de datos SQL.

La base de datos contiene dos tablas, una que guarda contenido de los coches y otra que guarda contenido de los compradores de esos coches.

Tenemos botones para Guardar, Actualizar y Eliminar datos de la base de datos, aparte de cambiar de tabla.

Más abajo encontramos una tabla, donde se nos muestra el contenido de la base de datos y podemos seleccionar una fila para editar o eliminar los registros, haciendo uso de los botones de abajo.
